Dungeon Turns
    1. Mark Ticks (to track time, resources, etc.)
    2. Make an encounter check.
    3. Declare actions.
    4. Make Perception-type checks as necessary.
    5. Resolve actions and track movemnet. 

Encounter check
    -Roll 1d8. 1 = Encounter
    -If encounter, roll on encounter table.
    -Determine encounter distance. 2d6 x 10 feet.
    -Make a reaction check.
    -Determine surprise if any. 

Reaction table
    2D6     Reaction
    2-3     Immediate Attack
    4-5     Hostile
    6-8     Cautious/Threatening
    9-10    Neutral
    11-12   Amiable
